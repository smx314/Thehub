# Thehub: 校园与生活高效管家

## 项目简介
Thehub 是一款为学生和上班族设计的简单工具。它有“任务助手”与“花销面板”两大功能，通过使用这个应用可以解决日常行程规划、财务记录的问题，满足个人专注能力提升的需求。

## 核心功能
1. **多用户数据隔离**：
    - **物理隔离**：支持多账号登录，所有数据（账单、日程、专注记录）均基于唯一用户 ID 进行严格过滤，确保多用户环境下数据的独立与隐私。
2. **财务管理模块 (Finance Master)**：
    - **极速记账**：内置自定义数字键盘，支持收入与支出的快速录入。
    - **动态统计**：实时计算总收入、总支出及结余，支持按日期范围筛选账单。
3. **校园助手模块 (Campus Assistant)**：
    - **多维日程**：提供“今日”、“明天”、“本周”、“年度”四个维度的日程视图，支持全功能的修改。
    - **云端同步**：集成 Retrofit 网络库，具备从指定远程服务器同步最新日程的能力，确保多设备数据一致。
4. **专注工具箱 (Focus Timer)**：
    - **精确计时**：以“秒”为单位的实时专注计时。
    - **专注统计**：自动统计专注次数与累计时长。
    - **报告导出**：支持选择特定日期范围生成专注报告，并可一键复制到剪贴板。

## 技术亮点
- **架构设计**：采用标准的 **MVVM (Model-View-ViewModel)** 模式，确保业务逻辑与 UI 层的彻底解耦。
- **本地存储**：基于 **Room Database** 实现高性能的本地持久化存储。
- **网络通信**：使用 **Retrofit + Gson** 进行 RESTful API 通信，支持异步数据加载。
- **健壮性优化**：
    - 针对 Fragment 生命周期进行了防御性编程，通过 Null Safety 检查有效防止视图销毁后的崩溃。
    - 利用 `switchMap` 和 `MediatorLiveData` 优化数据流分发，解决数据源切换时的潜在泄漏问题。
- **状态管理**：通过全局 `UserManager` 统一管理登录状态与用户标识。

## 技术栈
- **开发语言**：Kotlin
- **架构模式**：MVVM
- **数据库**：Room Database
- **网络请求**：Retrofit + OkHttp + Gson
- **UI 框架**：Material Design Components
