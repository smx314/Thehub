# Thehub 使用说明文档 (User Manual)

欢迎使用 **Thehub** —— 您的校园与生活高效管家！本应用集成了财务管理、行程规划与专注工具，助您轻松掌控大学生活。

---

## 🚀 核心功能介绍

### 1. 首页看板 (Dashboard)
首页是您的“信息中枢”，通过顶部的标签页可以轻松切换视图：
- **今天 (Today)**：实时展示最紧急的动态，包括今日日程和重要倒数日。
- **我的一周 (My Week)**：以时间线形式展示本周的每一天，方便您进行周计划管理。
- **年度计划 (Yearly)**：提供年度宏观视角，按月份选择日期查看详细计划。
- **日程管理**：点击右下角的“+”按钮，即可快速添加新的课程或倒数日提醒。支持点击条目进行编辑或删除。

### 2. 极简财务管理 (Finance)
告别复杂的记账流程，Thehub 提供极速记账体验：
- **高效录入**：点击右下角“+”号，弹出**全新设计的圆润数字键盘**。大按键设计与清晰的间距，确保单手录入金额快、准、稳。
- **视觉辨识**：系统采用鲜明的色彩区分账目——**红色表示收入**，**绿色表示支出**，让您的资金流向一目了然。
- **灵活筛选**：支持通过顶部的日期选择器筛选特定时间段的账单，自动统计该时段的总收入、总支出及结余。
- **分类统计**：提供餐饮、购物、交通等多种分类，帮助您精准分析消费习惯。

### 3. 智能专注计时 (Focus)
助您摆脱手机干扰，进入深度学习状态：
- **精确计时**：以**秒**为单位的实时倒计时，让每一秒的努力都清晰可见。
- **专注统计**：重新定义的专注逻辑——点击“开始”并最终点击“完成”即计入一次成功专注。
- **报告导出**：支持选择自定义日期范围生成专注报告，一键复制报告内容，方便分享或记录到手帐。

### 4. 个人中心与同步 (Profile)
- **多用户隔离**：支持账号登录。不同用户的数据（账单、日程、专注）完全物理隔离，保护您的隐私。
- **云端同步**：支持从远程服务器同步日程数据，确保您的计划在多设备间保持同步。

## 🛠️ 如何开始使用

### 第一步：登录与同步
1. 首次使用请在登录界面输入账号。
2. 进入“个人中心”可以查看当前登录状态。
3. （高级功能）日程支持从云端同步，确保您的课表永远是最新的。

### 第二步：高效记账
1. 切换到“财务”标签页。
2. 点击悬浮按钮，在弹出的对话框中通过数字键盘输入金额。
3. 选择收支类型（注意红色为收入，绿色为支出）和分类。
4. 点击“OK”即可保存。

### 第三步：专注训练
1. 切换到“专注”标签页。
2. 点击“开始专注”开启计时。
3. 完成学习任务后点击“完成”，系统将自动为您记录本次专注时长。

## 💡 技术亮点
- **极致稳定**：针对 Fragment 生命周期进行了深度优化，通过全方位的 Null Safety 检查，彻底告别意外退出。
- **流畅交互**：基于最新的 **Android Jetpack** 组件开发，采用 **MVVM** 架构，UI 响应零延迟。
- **全能适配**：完美支持系统级**深色模式**。在深色环境下，数字键盘背景会自动切换，确保按键清晰、护眼。
- **数据安全**：核心数据存储在本地 **Room 数据库**中，即便没有网络也能正常使用所有核心功能。
